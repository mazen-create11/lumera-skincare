## ✅ Corrections appliquées (26/06/2026)

Workflow de 13 agents (1 par fichier) + finitions manuelles. Sauvegarde des fichiers d'origine effectuée avant édition.

### Direction artistique
- **Fraunces réellement chargée** sur toutes les pages + `--serif` ET `--font-serif` remappés → la vraie serif éditoriale est active sur la home (avant : titres en Hanken Grotesk penché). Commentaire de design system mis à jour.
- **Footer de `about.html` réparé** (ajout `class="site-footer"` — le texte clair était invisible sur fond crème).
- **Pastille 840 nm** : violet `#8B5CF6` → rose de marque `#C97E8C`.

### Crédibilité / copy
- Longueurs d'onde : `633/830 nm` → **`630/840 nm`** (cohérence avec 630/840/1072).
- **Stat « 470 nm acné » retirée** (home + about) — le masque n'émet pas de bleu.
- Origine : « ingénierie suisse » → « ingénierie française et européenne ».
- 1072 nm : allégation « gènes / tissus adipeux » → formulation cosmétique recevable.
- Sérum : comparaison Rejuran/injectable nuancée.
- Fautes : `modélateurs`→`modulateurs`, `multi-stridale`→`multi-niveaux`, `Immédial`→`Immédiate`.
- Tag étude `830nm` → `proche infrarouge` ; carte « Instagram » de contact → « TikTok » (le lien pointait déjà vers TikTok).

### Conversion / UX
- **Prix barré 470 €** (= prix concurrent) → « **vs 470-600 € chez les leaders** » (non barré, conforme Omnibus).
- **Double barre sticky** « Ajouter au panier » retirée sur les 3 pages produit (gardé `.satc`).
- **Noms panier unifiés** (dédup) : `Masque LED Face + Cou` / `Sérum PDRN Glass Skin` / `Stylo Micro-needling` sur index, fiches et tiktok.html → fini les lignes en double.
- Verrou de scroll du fond à l'ouverture du panier.
- Étoiles : aria-label corrigé en « Noté 4,7 sur 5 ».

### Accessibilité (94 → 98)
- **Contraste corrigé** : eyebrows roses, liens/copyright du footer, micro-légendes (le `color-contrast` WCAG passe désormais).
- **Zoom mobile rétabli** (suppression de `maximum-scale=1.0`) sur les 3 fiches produit.
- **Skip-link** « Aller au contenu » + bloc `prefers-reduced-motion` global.
- `<main id="contenu">` + `autocomplete` sur le formulaire de contact.

### SEO / perf
- **Open Graph + Twitter Cards** ajoutés/rendus **absolus** sur toutes les pages → aperçus de partage fonctionnels (canal TikTok).
- **JSON-LD Organization + WebSite** sur la home (sameAs TikTok).
- JSON-LD Product : URLs absolues + `priceValidUntil`.
- **Meta descriptions** ajoutées (contact, livraison, legal).
- `bundle.html` : preload corrigé (513 Ko inutiles → vraie image LCP de 80 Ko).
- Performance home **→ 100**.

## Détail des problèmes détectés (référence)

**CRO** — checkout mailto (critique), badges paiement/3×/abonnement non encaissables, double sticky, prix barrés non réels, étoiles 5/5 pour 4,7, CTA home sous l'image en mobile, preuve sociale absente des pages produit.

**Copy** — garantie contradictoire, 470 nm acné, longueurs d'onde 633/830, origine suisse/Asie/France, fautes, jargon pharma vs voix premium, surpromesses 1072 nm / Rejuran, formats de chiffres incohérents, microcopy « satisfait·e » en 3 variantes.

**DA/UX** — Fraunces absente, footer about cassé, 2 design systems parallèles (rose en 4 valeurs), boutons/headers/footers divergents, dot 840 nm violet.

**SEO** — OG absent 7/9 pages + og:image relative, pas d'Organization/WebSite, pas de BreadcrumbList, meta desc manquantes, favicon absent 5 pages, JSON-LD images relatives. (Faux positif écarté : les `alt` produits sont en fait corrects.)

**A11y** — contraste eyebrows/footer, zoom bloqué, reduced-motion ignoré, pas de skip-link, modales sans role=dialog/focus-trap, vignettes galerie non clavier, heading-order.

**Tech** — dédup panier cassée, double sticky, about.html déconnecté, scripts.js legacy concurrent, checkout mailto, GA4/Meta pixel = placeholders, ancres mortes, code vidéo mort.

**Perf** — bundle preload erroné, pas de srcset, CSS/fonts non optimisés, ~26 Mo d'artefacts non chargés en prod (à gitignore).
