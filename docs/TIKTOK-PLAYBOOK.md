## 0. LA MÉTHODE (à internaliser — c'est le moteur)

1. **Le taux de complétion est LA métrique reine** (facteur de classement n°1 du FYP, confirmé par TikTok). Tout — durée, hook, montage, boucle — sert à le maximiser. Les **saves et shares** pèsent bien plus que les likes.
2. **Le hook se joue en 0-3 s, avec du TEXTE À L'ÉCRAN dès la frame 1** (85 % regardent sans le son). Jamais d'intro « coucou les filles ». Le mot-clé cible doit être **dit ET écrit** dans les 3 premières secondes.
3. **TikTok = moteur de recherche** (~40 % de la Gen Z y cherche avant Google). Il transcrit tout l'audio → TikTok SEO = mot-clé dans le hook parlé + texte écran + légende + nom de profil. Le SEO capitalise dans le temps (rankings stables).
4. **Vendre par PROBLÈME, pas par marque** (logique « discover & learn »). Chaque vidéo s'ouvre sur la préoccupation peau (terne, rides, acné, barrière), jamais sur la marque ni le prix.
5. **Durée selon l'objectif** : 11-18 s = acquisition/viralité (boucle auto, complétion > 100 %) ; 21-34 s = storytelling ; 30-60 s = éducatif/démo (nourrit le SEO audio).
6. **Montage** : jump cuts serrés (zéro temps mort) + changement visuel toutes les 5-10 s + sous-titres animés obligatoires.
7. **Signaux marché FR 2025-2026** (rapport Spate) : barrière cutanée/hydratation en explosion (panthénol +338 %, centella +263 %, masques collagène +218 %). Catégorie masques LED : **+400 % sur TikTok Shop**, dont ~26 % des ventes en **LIVE Shopping**.
8. **High-ticket à 249 €** = cycle de 7-12 points de contact. On ne juge jamais une vidéo isolée sur sa vente directe : on nourrit (TOFU → MOFU → BOFU).

## 7. SOP QUOTIDIENNE (routine répétable)

1. **Veille (la veille, 10 min)** — Repérer 1 son montant cohérent DA rose (Creative Center) + 1 requête longue-traîne (autocomplétion TikTok). Choisir le sujet (problème → onde).
2. **Script (5 min)** — Écrire UNIQUEMENT le hook 0-3 s (texte écran 1re seconde + 1re phrase) avec le mot-clé. Structure : hook concern-led → preuve/démo → mini-CTA. 3 variantes de hook.
3. **Tournage (téléphone, lumière naturelle)** — Ouvrir sur le moment fort. 3-5 prises courtes + B-roll. **Batcher 5-10 vidéos le week-end.**
4. **Montage CapCut (15-20 min)** — Jump cuts serrés, changement visuel toutes les 5-10 s, sous-titres animés (mot-clé orthographié dedans). Clips <18 s : boucler la fin sur le début.
5. **Publication** — Légende : mot-clé en début + 1 question pour les commentaires. Hashtags : **3-5 max** (1 large + 2-3 niche FR + 1 marque). **JAMAIS #fyp #pourtoi #viral** (signal spam). Poster ~30 min avant le pic.
6. **Engagement (60 premières min = critiques)** — Maria répond à TOUS les commentaires dans l'heure. Épingler le commentaire-CTA. Noter les nouvelles questions.
7. **Analyse (J+1, 5 min)** — Courbe de rétention : chute à 3 s → refaire le hook ; chute au milieu → resserrer les cuts. Noter vues / complétion / saves / abonnés.
8. **Hygiène anti-shadowban** — Parler « éclat / confort / aspect de la peau », éviter « soigne / traite / guérit ». Varier les hashtags. 1 jour off/semaine. Chute brutale de 80-90 % → pause 48-72 h, reprendre propre.
